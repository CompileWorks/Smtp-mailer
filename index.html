Instructions

  1. Create your HTML email template in any code editor
  2. Save it as an HTML file (.html or .htm extension)
  3. Upload it using the "Upload HTML" tab
  4. Give it a descriptive name and optional description
  5. Select your template from the "My Templates" tab to preview it
  6. Use the "Send Email" tab to send your HTML email to recipients
  7. All templates are stored locally in your browser
